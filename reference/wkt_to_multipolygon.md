# wkt_to_multipolygon

wkt_to_multipolygon

## Usage

``` r
wkt_to_multipolygon(v, call = rlang::caller_env())
```

## Arguments

- v:

  The WKT to convert into a multipolygon.

- call:

  The env from which this was called (defaults to the direct calling
  environment).

## Value

The multipolygon equivalent of wkt.
