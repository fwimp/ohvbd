# Reimplementation of cli's progress style shim

Reimplementation of cli's progress style shim

## Usage

``` r
progress_style(x)
```

## Arguments

- x:

  a string to find in
  [`cli::cli_progress_styles()`](https://cli.r-lib.org/reference/cli_progress_styles.html).

## Value

a formatting list from
[`cli::cli_progress_styles()`](https://cli.r-lib.org/reference/cli_progress_styles.html).
