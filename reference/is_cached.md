# Check whether an object has been loaded from cache by ohvbd

Check whether an object has been loaded from cache by ohvbd

## Usage

``` r
is_cached(x)
```

## Arguments

- x:

  The object to check.

## Value

A boolean indicating whether an object has been loaded from the cache.

## Author

Francis Windram

## Examples

``` r
if (FALSE) { # interactive()
is.cached(c(1,2,3))
}
```
