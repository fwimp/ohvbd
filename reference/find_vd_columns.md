# Get current columns in VecDyn datasets

Get all the current column names in VecDyn, alongside associated data if
desired.

## Usage

``` r
find_vd_columns(full = FALSE, basereq = vb_basereq())
```

## Arguments

- full:

  whether to return all the data about current fields, or else just
  return the names

- basereq:

  an [httr2 request](https://httr2.r-lib.org/reference/request.html)
  object, as generated by
  [`vb_basereq()`](https://ohvbd.vbdhub.org/reference/vb_basereq.md). If
  `NA`, uses the default request.

## Value

A character vector (or dataframe) of column information.

if (FALSE) { # interactive()
