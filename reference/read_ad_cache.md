# Read AREAdata from cache file

Read AREAdata from cache file

## Usage

``` r
read_ad_cache(path, metric, gid, warn = TRUE)
```

## Arguments

- path:

  cache path.

- metric:

  metric to retrieve.

- gid:

  gid to retrieve.

- warn:

  Whether to warn if a cached file is older than 6 months.

## Value

cached data.
