# Read AREAdata from cache file

Read AREAdata from cache file

## Usage

``` r
read_ad_cache(metric, gid, path = NULL, warn = TRUE)
```

## Arguments

- metric:

  metric to retrieve.

- gid:

  gid to retrieve.

- path:

  cache path.

- warn:

  Whether to warn if a cached file is older than 6 months.

## Value

cached data.
