# Fetch specified data from a set of ids

This is a convenience method that infers and applies the correct fetch
function for the input ids.

## Usage

``` r
fetch(ids, ...)
```

## Arguments

- ids:

  An object of type `ohvbd.ids` (generated from a search, manually
  packaged using
  [`ohvbd.ids()`](https://ohvbd.vbdhub.org/reference/ohvbd.ids.md) or
  generated by another function).

- ...:

  Any other arguments to be passed to the underlying fetch functions
  (see [`fetch_vt()`](https://ohvbd.vbdhub.org/reference/fetch_vt.md)
  and [`fetch_vd()`](https://ohvbd.vbdhub.org/reference/fetch_vd.md) for
  specific arguments).

## Value

The downloaded data, as an `ohvbd.responses` object.

## Examples

``` r
if (FALSE) { # interactive()
search_hub("Ixodes", "vt") |> fetch()
}
```
